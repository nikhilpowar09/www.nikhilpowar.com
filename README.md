# 🚀 Nikhil Powar — Portfolio Website

A modern and interactive personal portfolio website built using **React**, **Vite 5**, and **Framer Motion** to showcase projects, skills, experience, and creative web development work.

---

## ✨ Features

- ⚡ Fast performance with Vite 5
- 🎨 Smooth animations using Framer Motion
- 📱 Fully responsive design
- 🌙 Modern UI/UX
- 🚀 Optimized production build
- 🔥 Interactive portfolio sections
- 📂 Project showcase
- 📧 Contact section

---

## 🛠️ Tech Stack

- **React**
- **Vite 5**
- **Framer Motion**
- **JavaScript**
- **HTML5**
- **CSS3**

---

## 📦 Requirements

- Node.js **>= 18**
- npm or yarn

---

# 📥 Installation

Clone the repository:

```bash
git clone https://github.com/nikhilpowar09/www.nikhilpowar.com.git
```

Move into the project folder:

```bash
cd www.nikhilpowar.com
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Local Development

Start the development server:

```bash
npm run dev
```

The app will run on:

```text
http://localhost:5173
```

---

# 🏗️ Production Build

Create an optimized production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
