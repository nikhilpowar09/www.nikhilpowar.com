# Nikhil Powar — Portfolio

Personal portfolio built with **React + Vite 5 + Framer Motion**.

> ✅ Compatible with Node.js **≥ 18** (including v20.18)

## Local Development
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import repo
3. Framework: **Vite** | Build: `npm run build` | Output: `dist`
4. Click Deploy 🚀
